## Codebase Patterns
- Landing uses Next.js 15 with App Router in `landing/`
- Fonts loaded via `next/font/google` with CSS variable strategy (--font-syne, --font-azeret, --font-dm)
- Design system colors are CSS custom properties in globals.css — never use Tailwind color utilities
- Badge pattern: `[ LABEL ]` in Azeret Mono, 11px, all-caps, letter-spacing 0.12em
- No Tailwind config — use raw CSS variables for all theming
- Staggered animations: use `.fade-up` class + inline `animationDelay` style
- `.btn-primary` and `.btn-ghost` are shared button classes in globals.css
- No lint script — `tsc --noEmit` is the only quality gate
- Client components ("use client") go in `landing/app/` alongside page.tsx
- Terminal animation uses setTimeout chain with cleanup via clearTimeout in useEffect return
- Shared section styles: `.section-label` and `.section-title` for consistent section headers
